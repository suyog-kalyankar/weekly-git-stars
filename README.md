# Weekly Git Stars

Welcome to the Weekly Git Stars application! 

## Overview

Weekly Git Stars is a web application that provides a curated list of popular repositories from the last week using a public API. You can easily clone the repository, install dependencies, and run the application to explore and star repositories of your interest.

## Getting Started

Follow these steps to set up and run the application on your local machine:

1. Clone the repository:
```git clone https://github.com/suyog-kalyankar/weekly-git-stars.git```

2. Install dependencies:
```cd weekly-git-stars```
```npm install```

3. Run the application:
```npm start```
The application will be available at http://localhost:3000.

## Features

 - Popular Repositories: The application fetches popular repositories from the last week using a public API.
 - Star Repositories: Users can manually star repositories, and the application stores these starred repositories in the local storage for easy retrieval.
 - Filter Starred Repositories: Utilize radio buttons to filter and view only the repositories you have previously starred.
 - Search Filter: Use the language search filter to find repositories based on the programming language.
 - Responsiveness: The application is designed to be responsive, providing a seamless experience across various devices and screen sizes.
